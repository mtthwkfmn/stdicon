stdicon

If you've ever had a need for filetype icons, just hotlink ours. You can use a file extention like /html, a mimetype like /application/pdf and/or choose from a specific set like /silk/html or even combine them like /silk/image/png. You can also use the ?size=16 to get a consistent size of icons, and the ?default=http%3a%2f%2fmetaward.com%2ffavicon.ico to force a default instead of an HTTP 404.

Quick Examples

  • /html
  • /crystal/html
  • /crystal/text/html
  • /crystal/text/html?size=16
  • /garbage?size=16&default=http%3a%2f%2fstdicon.com%2fhtml
